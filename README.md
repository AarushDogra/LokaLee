# Lokaee


Lokalee — Travel Global. Live Local.

Lokalee is a smart travel companion that helps travelers experience any city the way locals do — from real-time weather and budget planning to local food, attractions, and a backpacker survival guide.

Whether you're a budget backpacker or a curious explorer, Lokalee helps you drift anywhere and belong everywhere.



 API | Purpose |
|---|---|
| OpenWeatherMap | Real-time weather by city || OpenTripMap | Local attractions & landmarks || RestCountries | Country info, currency, languages || ExchangeRate API | Live budget currency converter || TheMealDB | Local cuisine, recipes & food culture |




Planned Features

 🏠 Home
- Search any city and instantly see weather + country info
- Food of the Day (random local meal)
- Quick jump to all sections

 💰 Budget Planner
- Enter total budget and trip duration
- Daily spend calculator with currency conversion
- Attractions filtered by free / cheap / paid

 ✈️ Destination Comparison
- Compare 2–3 cities side by side
- Weather, cost level, attraction count
- Sort by best weather, most attractions, cheapest

 🍜 Food & Culture Explorer
- Local restaurants from OpenTripMap
- Traditional recipes from TheMealDB by country
- Filter by cuisine, category, rating

 🎒 Backpacker Survival Guide
- Auto checklist based on destination weather
- Local currency, language, emergency info
- Save and persist checklist with localStorage

 ⚙️ General
- Search, filter, sort using Array HOFs
- Favorite cities saved with localStorage
- Dark / Light mode toggle
- Pagination for attractions and food
- Debounced search input
- Loading indicators on all API calls
- Fully responsive (mobile, tablet, desktop)
